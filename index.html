Projetos em Ciência de Dados

Clusterização para Programa de fidelidade "Insiders"

Uma empresa de e-commerce deseja implantar um programa de fidelidade para o grupo clientes mais valiosos chamado "Insiders".
Para isso, um modelo de Machine Learning não supervisionado foi utilizado para a clusterização dos clientes, agrupando-os por similaridade.
O grupo de clientes especiais encontrado representa 15.7% do total de clientes da base, e é responsável por 51.7% do faturamento.

As ferramentas utilizadas foram:

  • Git e Github.
  • Python, Pandas, Numpy, Seaborn.
  • ML libs: Sklearn, Scipy.
  • ML Models: K-Means, Hierachical Clustering, Gaussian Mixture Model, DBSCAN.
  • Métricas de avaliação: WSS e Silhouette Score.
  • Deploy: AWS Cloud (S3, EC2 and RDS).
  • Visualização de Dados: Google Looker Studio.
